# AI-Powered Chatbot

This is an intelligent chatbot project built using Python, Flask, and Hugging Face Transformers. It provides contextual responses to user input and logs all interactions in a local SQLite database.

## Technologies Used
* **Python**
* **Flask** (Web Framework)
* **Transformers** (AI Model: DistilGPT2)
* **SQLite** (Database for interaction logs)

## How to Run
1. Clone this repository.
2. Install dependencies: `pip install flask transformers torch nltk`
3. Run the application: `python app.py`
4. Access the chatbot in your browser at `http://127.0.0.1:5000`
